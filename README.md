# 机器觉醒 (Machines Awakening)

> **itch.io 2026 GameJam 参赛作品**

一款基于 Godot 4 开发的像素风格潜入黑客游戏。

---

## 一、项目简介

玩家扮演一名黑客，在充满机械与工人的工厂中潜行，通过黑入各种机器设备、解锁新能力来完成目标。合理利用能量、躲避安保与监控，寻找通关路径。

---

## 二、功能特性

- **玩家系统**：黑入能力、能量值管理、能力解锁
- **可交互机器**：机械臂、监控探头、传送带、门禁控制器、剧情终端
- **NPC 行为**：工人（巡逻/追踪）、安保（更警觉）
- **核心系统**：存档、关卡切换、对话管理、JSON 关卡加载
