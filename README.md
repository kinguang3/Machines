# Robot Battle Arena

一款基于 Godot 4 开发的机器人战斗竞技场游戏。

---

## 一、项目简介

玩家可以组装自定义机器人，通过图形化编程界面为机器人编写战斗指令，然后在竞技场中与 AI 对手进行战斗。

---

## 二、核心功能

### 🤖 机器人组装
- 收集和选择不同的机器人部件
- 自定义机器人的能力和属性

### 🧠 图形化编程
- 拖拽式指令块编程
- 支持条件判断和循环
- 实时指令预览

### ⚔️ 竞技场战斗
- AI 对手对战系统
- 战斗日志记录
- 回合制战斗机制

---

## 三、项目结构

```
robot_battle_arena/
├── assets/          # 原始资源
│   ├── art/         # 像素画素材
│   └── audio/       # 音效和音乐
├── game/            # Godot 项目
│   ├── scenes/      # 场景文件
│   ├── scripts/     # 游戏脚本
│   └── resources/   # 数据资源
└── README.md
```
